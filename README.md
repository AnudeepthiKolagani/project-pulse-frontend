# Project pulse 

A modern web application for Employee Management and Project Tracking

## Tech Stack

- React
- REST API Integration
- Modern UI

## Getting Started

### Prerequisites

- Node.js 
- Yarn or NPM

Check your Node version:

```bash
node -v
```

## Installation

### 1. Clone the repository

```bash
git clone <your-repo-url>
cd <project-folder>
```

### 2. Install dependencies

```bash
yarn install
# or
npm install
```


## Run the Development Server

```bash
npm run dev
```

Open your browser and navigate to:

```
http://localhost:3000
```
